스켈레톤이란? 
- 웹 페이지에서 로드시간이 짧은것처럼 보이게 하는 방법 중 하나.
- 데이터가 로드되기 전에 콘텐츠의 자리비움(placeholder)를 표시해서 사용자가 덜 지루하게 느끼게 해준다.

- 장점
  : 유저들에게 인지된 로딩시간이 더 빠르게 느껴지게 해준다.
- 단점
  : 스켈레톤을 화면마다 표시해야하기 떄문에 상대적으로 시간이나 비용이 많이든다.

- ex)
  : 페이스북, 링크드인, 유튜브, 구글드라이브 등등..

- 규칙
  : 콘텐츠의 로드를 방해하면 안된다. 
  디자인 할 때 애니메이션을 사용하는것이 좋다. purse보다 wave추천
